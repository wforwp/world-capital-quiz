# 나라별 수도 맞추기 게임 (일반 웹 버전)

일반 웹 브라우저에서 실행 가능한 나라별 수도 맞추기 퀴즈 게임입니다.

## 주요 기능

- **난이도 선택**: 쉬움, 보통, 어려움 3가지 난이도
- **Combo 시스템**: 연속 정답 시 점수 배율 증가
- **하트 시스템**: 3개의 하트로 생명 관리
- **스피드 모드**: 빠른 게임 플레이 옵션
- **최고 점수**: 로컬 스토리지에 최고 점수 저장
- **도움말**: 게임 방법 및 팁 제공

## 설치 및 실행

### 로컬 개발 환경에서 테스트하기

```bash
# 의존성 설치
npm install
# 또는
bun install

# 개발 서버 실행 (로컬 테스트)
npm run dev
# 또는
bun run dev

# 브라우저에서 http://localhost:5173 (또는 터미널에 표시된 주소) 접속
# 코드 변경 시 자동으로 새로고침됩니다
```

### 프로덕션 빌드

```bash
# 프로덕션 빌드
npm run build
# 또는
bun run build

# 빌드 결과 미리보기
npm run preview
# 또는
bun run preview
```

**참고**: GitHub에 푸시하기 전에 반드시 로컬에서 `npm run dev`로 테스트해보세요!

## 기술 스택

- React 19
- TypeScript
- Vite
- CSS3

## 토스 앱뷰인토스 버전과의 차이점

- 토스 앱뷰인토스 프레임워크 의존성 제거
- SafeAreaInsets 관련 코드 제거
- 리더보드 기능을 로컬 스토리지 기반 최고 점수로 대체
- backEvent 리스너 제거 (일반 브라우저 뒤로가기 사용)
